# chainlit-bedrock-base
Chainlit Bedrock Base Code

##### Install

pip install --upgrade boto3
pip install --upgrade langchain-community
pip install --upgrade chainlit
pip install --upgrade python-dotenv

##### Launch Locally

chainlit run app.py -h
